# Universo MGO

O Universo MGO é uma plataforma de aventuras outdoor para pais e filhos.

Nosso propósito é transformar lugares reais em desafios memoráveis.

As missões são construídas pela combinação de:

- Estilo
- Arquitetura
- Narrativa
- Cenário

## Pilares

- A família é sempre protagonista.
- O personagem é sempre o mentor.
- O desafio é o produto.
- A história cria contexto.
- O Método CODE estrutura toda a experiência, mas permanece invisível para a família.
