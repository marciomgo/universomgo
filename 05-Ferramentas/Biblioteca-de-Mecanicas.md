# Biblioteca de Mecânicas

Uma mecânica é a menor unidade de design de uma missão do Universo MGO.

Ela não representa uma história nem um desafio específico.

Ela representa uma forma de interação entre os participantes e a experiência.

Cada mecânica deve ser catalogada contendo:

- Nome
- Objetivo
- Descrição
- Como funciona
- O que exige dos participantes
- O que desenvolve nos participantes
- Intensidade física
- Intensidade cognitiva
- Intensidade de cooperação
- Faixa etária sugerida
- Duração média
- Ambientes em que pode ser aplicada
- Narrativas com as quais combina
- Exemplos de aplicação
- Processos Cognitivos estimulados

Arquiteturas em que pode ser utilizada

As missões são construídas pela combinação de diferentes mecânicas, organizadas por uma narrativa comum e orientadas pelos princípios do Método CODE.
